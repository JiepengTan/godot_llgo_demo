# libgodot llgo demo

this repo is base on the [libgodot](https://github.com/migeran/libgodot_project): 

## windows
### build libgodot 
```
build_libgodot.bat
```


### build sample 
```
cd samples/cpp_sample
mkdir build
cmake -S . -B build -G Ninja
cd build
ninja
```

### run sample
```
sample.exe
```
